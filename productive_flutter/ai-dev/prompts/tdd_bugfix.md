1. **Root cause analysis** (trace exact failure path)
2. **Repro test first** (test should fail before fix)
3. **Minimal fix** (smallest change to pass)
4. **Regression tests** (cover edge cases)
5. **Verify** (tests pass; summarize files changed)

STOP after each step and wait for approval.
